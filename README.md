# Expiration Date Management
<a href="http://koichi73.pythonanywhere.com/" target="_blank">Expiration-Date-Management</a>は、食材の賞味(消費)期限を管理するためのWebアプリケーションです。フードロスを削減するために作成しました。

## Features
- ユーザーログイン
- レシート読み取り(OCR)
- 賞味(消費)期限の手動登録、編集、削除
- レシピ検索
